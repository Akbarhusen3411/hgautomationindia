== Description ==
HG Automation - Industrial Automation Company Theme
A modern, responsive one-page theme for industrial automation businesses.

== Installation ==

1. Download or copy the entire "hg-automation" folder

2. Upload to your WordPress site:
   - Via FTP: Upload to /wp-content/themes/
   - Via WordPress Admin:
     a. Zip the "hg-automation" folder
     b. Go to Appearance > Themes > Add New > Upload Theme
     c. Upload the zip file and click "Install Now"

3. Activate the theme:
   - Go to Appearance > Themes
   - Find "HG Automation" and click "Activate"

4. Set Homepage:
   - Go to Settings > Reading
   - Select "A static page"
   - Set "Homepage" to any page (the theme uses front-page.php)
   - Save Changes

5. Upload Your Logo:
   - Go to Appearance > Customize > Site Identity
   - Click "Select logo" and upload your logo
   - Publish changes

== Theme Features ==

- Responsive design for all devices
- Fixed header with smooth scroll navigation
- Animated hero section with company branding
- Services showcase with hover effects
- About section with company statistics
- Contact form with email notifications
- Social media integration
- Modern CSS animations

== Customization ==

1. Company Information:
   - Go to Appearance > Customize > Company Information
   - Update phone, email, and address

2. Social Media Links:
   - Go to Appearance > Customize > Social Media Links
   - Update LinkedIn, Twitter, Facebook URLs

3. Logo:
   - Go to Appearance > Customize > Site Identity
   - Upload a custom logo

== Contact Form ==

The theme includes a built-in contact form that sends emails to your WordPress admin email.
To change the recipient email, go to Settings > General > Administration Email Address.

For advanced contact forms, install Contact Form 7 plugin - the theme is styled to work with it.
